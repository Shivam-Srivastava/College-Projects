INSTRUCTIONS TO COMPILE AND RUN THE CODE
-------------------------------------------------------------------------
Instructions to run 3D reconstruction:
1. Enter folder 'stereo'
2. chmod +x script.sh
3. ./script.sh

Instructions to run hair reconstruction from image:
1. Enter folder 'hair'
2. chmod +x script.sh
3. ./script.sh
